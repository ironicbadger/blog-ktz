We both like to collect things. We've visited so many States now we figured it was time to devise a system to track them.

It's actually a bit more complex than you might imagine. Yes, visiting States sounds like a simple construct but there are different levels of visitation. Living in a State shouldn't count the same as driving through one now should it? Thus, our visitation matrix was born:

| Platinum | Gold | Silver | Bronze |
| --- | --- | --- | --- |
| Lived there | Stayed 5+ nights | Stayed 2+ nights | Driven through |
| Born there | OR | OR | \* Flyovers / layovers not valid |
|  | 2 silver trips | Eaten there (not fast food) |  |
